# all in one script

## ダウンロード

### キャッシュフリー
```sh
wget -q -O /tmp/aios.sh "https://raw.githubusercontent.com/site-u2023/aios/main/aios.sh?cache_bust=$(date +%s)"; sh /tmp/aios.sh
```
